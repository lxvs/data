Introduction

Johnny's Git Kit, or jg, is a POSIX compliant shell script providing several enhancements for git usage, including:

  • Create a GitHub repository in command line interface!
  • Better views of git log!
  • Create a tarball with all submodules!
  • Diff with Beyond Compare!
  • Resolve merge conflicts with Beyond Compare!
  • Query a git commit using a Gerrit Change-Id pattern!
  • Push local commits to Gerrit without typing refs/for/!
  • Do something for each git repository in a directory!
