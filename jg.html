Introduction

Johnny's Git Kit, or jg, is a bash script providing several handy commands for git, including

  • Do something for all git repositories in current directory
  • Diff with beyond compare Only supported in Windows for now.
  • merge conflicts with beyond compare Only supported in Windows for now.
  • Get a git commit by Gerrit Change-Id For details, see Gerrit Code Review - Change-Ids
  • Push commits to Gerrit
  • A better view of git log!
  • Create a GitHub repository in command line!
