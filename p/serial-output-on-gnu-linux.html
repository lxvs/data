Enable Serial/SOL Output on GNU/Linux Systems

Oct. 6, 2021. Updated on June 7, 2022.

Content

  • Temporarily
  • Permanently

Temporarily

When system boot to grub menu, press e to edit boot options, in line starting with linux ($root)/vmlinuz-, append console=ttyS0,115200n8 console=tty0 to the end. This is for most commonly used modern BIOS, but it should match your BIOS settings in case yours are different. Then press Ctrl + x to continue to boot.

Permanently

RHEL 8 is used in this example.

Step 1, enter directory /etc/default and make a backup of file grub. 

cd /etc/default
sudo cp grub grub.bak

Step 2, edit file /etc/default/grub, in line starting with GRUB_CMDLINE_LINUX=, append console=ttyS0,115200n8 console=tty0 within the quotation marks.

Step 3, execute below commands.

sudo grub2-editenv - unset kernelopts
sudo grub2-mkconfig -o /boot/efi/EFI/redhat/grub.cfg

If Something Is Wrong...

In case that something was did mistakenly in above procedure and the system cannot boot, you can try to boot into single user mode and restore the file grub from grub.bak created in first step, and then redo step 3.
