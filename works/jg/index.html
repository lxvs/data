Introduction

Johnny's Git Kit, or jg, is a POSIX compliant shell script providing several enhancements for git usage, including:

  • creating GitHub repositories in command line interface
  • better views of git log
  • creating tarballs including with submodules
  • diff using Beyond Compare
  • resolving merge conflicts with Beyond Compare
  • querying a git commit using a Gerrit Change-Id pattern
  • pushing local commits to Gerrit without typing `refs/for/'
  • doing something for each git repository in a directory
